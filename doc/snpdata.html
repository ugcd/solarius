Annotation for simulated genotypes in dat50 data set adapted from FFBSKAT R package

Usage

data(dat50)

Format

A data frame with 100 rows and 4 variables:

name
SNP name.

chrom
Chromosome.

position
Position in bp.

gene
Gene.

Description

Annotation for simulated genotypes in dat50 data set adapted from FFBSKAT R package

data(dat50)
