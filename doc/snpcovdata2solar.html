Function snpcovdata2solar.

Usage

snpcovdata2solar(mat, out, dir, nGTypes = FALSE)

Description

The function emulate the `snp load` SOLAR command. Two output files are produced: `snp.genocov` and `snp.geno-list`. The steps are the following: (1) add prefix `snp_` to SNP names; (2) (optiona) compute stats on # genotyped individuals (columns `nGTypes`); (3) write data and metadata into files.

Details

Example of `snp.genocov` file: id,nGTypes,snp_s1,snp_s2,... 1,50,0,0,... 2,50,0,0,...

Example of `snp.geno-list` file: snp_s1 snp_s2 ...
