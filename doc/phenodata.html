Phenotypes of dat50 data set adapted from FFBSKAT R package

Description

A mixture of unrelated and related individuals (a custom kinship matrix is given) were originally simulated to test methods of the variant-collapsing approach.

Details

This data set is used here to test the ability of SOLAR to work with a custom kinship matrix in both polygenic and association analyses.
