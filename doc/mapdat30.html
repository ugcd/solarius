Annotation for simulated genotypes in dat30 data set adapted from multic R package

Usage

data(dat30)

Format

A data frame with 100 rows and 4 variables:

SNP
SNP name.

chr
Chromosome.

pos
Position in bp.

gene
Gene.

Description

A hundred of synthetic SNPs were randomly generated for dat30 data set. Annotation also was generated to be plot the association results with Manhattan plot.
