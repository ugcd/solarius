Genotypes simulated for dat50 data set adapted from FFBSKAT R package

Usage

data(dat50)

Format

A matrix with 66 rows and 50 columns.

Description

50 synthetic SNPs were generated. The genotypes are coded in the format such as 1/1, 1/2 and 2/2.

Examples

data(dat50)

str(genodata)
