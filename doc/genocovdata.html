Genotypes as covariates in dat50 data set adapted from FFBSKAT R package

Usage

data(dat50)

Format

A matrix with 66 rows and 50 columns.

Description

50 synthetic SNPs were generated. A matrix of covariates was derived from the genotype data according to the additive model.
