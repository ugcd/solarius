Function solarKinship

Usage

solarKinship(df, dir, ...)

Description

The function (1) puts the data set df in SOLAR format, (2) separates it into two parts, pedigree and phenotypes, and then (3) expots both data sets in the directory dir, (4) read the specific `phi2.gz` file, where the kinship coefficients multiplied by 2 are stored by SOLAR.

Note

IDs in df are assumed to be not duplicated.
